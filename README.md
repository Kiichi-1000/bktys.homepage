# BKTYSSHOP Homepage

BKTYS（バスケットボール部）の公式ホームページです。

## ブランド

- **25s.** - 高級ブランドコレクション
- **Nuv.** - ユニセックスブランドコレクション  
- **Members** - 部員向けコレクション

## 機能

- レスポンシブデザイン（PC・タブレット・スマートフォン対応）
- 商品スライドショー
- ブログ機能
- BASEショップ連携
- 管理画面（将来のAPI連携用）

## 技術スタック

- HTML5
- CSS3
- JavaScript (ES6+)
- PHP (API連携用)

## セットアップ

1. リポジトリをクローン
2. Webサーバーでホスト
3. ブラウザでアクセス

## GitHub Codespaces

このプロジェクトはGitHub Codespacesで編集可能です。

1. GitHubリポジトリページで「Code」→「Codespaces」→「Create codespace」
2. ブラウザベースのVS Codeで編集開始
